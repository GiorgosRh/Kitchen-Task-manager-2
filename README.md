# Kitchen-Task-manager-2
Simplified task manager — connected to [BitChef](https://github.com/GiorgosRh/BitChef).

## BitChef Integration
Fetches live shift and worker data from the BitChef backend server (`helioskitch-mpdj7fvp.manus.space/api/kitchen`). The `overview.html` auto-refreshes every 30 seconds from the same API.
